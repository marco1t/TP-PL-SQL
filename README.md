# TP SQL - Base de données Voyageur

Ce projet contient les scripts SQL pour le TP sur la base de données Voyageur.

## Fichiers

*   `schema.sql` : Création des tables (Voyageur, Logement, Sejour, Activite) et insertion des données de test.
*   `reponses.sql` : Contient l'ensemble des requêtes SQL répondant aux questions du TP (1 à 55).

## Instructions

1.  Ouvrez votre outil de gestion de base de données (ex: DB Browser for SQLite, DBeaver, ou ligne de commande).
2.  Exécutez le script `schema.sql` pour créer la structure et les données.
3.  Vous pouvez ensuite exécuter les requêtes du fichier `reponses.sql` une par une pour voir les résultats.
